格子升级：当局流派与收益效率

「格子升级」指绑定在盘面 / 单格机制上、主要影响本局玩法走向与资源转化率的强化层（不同版本 UI 可能写作强化格、模组、祝福等）。在三大体系里，它承担流派核心角色。

命名说明

公开商店文案侧重「永久升级 + 一次性卡牌 + 修饰符 / 陷阱」；若你客户端中的「格子类强化」条目名称与本文用词不一致，以游戏内分类与描述为准，本文只描述策略层级。

它决定什么

  • 当局节奏：偏快清、偏稳健控图、偏情报扩张或偏经济转化，往往由这一层的抓取顺序定型。
  • 收益效率：同样一次清盘，流派不同会导致金币与进度的「边际产出」不同，进而影响商店购买力。

与永久线如何配合

经验上：永久线先买到不暴毙的阈值，再让格子升级去「放大优势」；若当局流派与当前盘面修饰符冲突，宁可保守抓取，留给 卡牌保险 与生命试错（风险篇）。

流派思考框架（不绑定具体词条）

情报扩张型
优先提高「每一下点击带来的约束信息量」，适合陷阱多、歧义多的阶段。
经济滚雪球型
优先提高金币或商店相关转化率，适合已能稳定存活、想拉长局的玩家。
攻坚 / Boss 特化型
在确认能稳定见到 Boss 后，再倾斜抓取，避免前期空转。
